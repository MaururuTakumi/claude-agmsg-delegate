Fable and ordinary Sonnet consultation are read-only advisory jobs. They may
inspect the current project directory with `Read`, `Glob`, and `Grep` in plan
mode. When the user asks Sonnet to implement a change, Sonnet may edit the
target Git workspace through the explicit `--workspace-write` mode with
`Read,Edit,Write,Glob,Grep`. Codex remains responsible for commands, tests,
integration, and the final decision; Bash is not available to delegated Claude.
